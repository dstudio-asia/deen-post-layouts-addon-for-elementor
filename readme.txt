== Description ==

Are you looking for a best post layouts addon for elementor wordpress plugin? Then Deen is the best and perfect post layouts addon for your WordPress website. Deen post layouts  addon is a wordpress plugin that helps you to make simple, clean and eye-cache layouts for posts. This addon comes with 4 layouts types: Standard Post Layout, Left Posts Layout, No Image Posts Layout and Single Column Posts Layout. You can easily Customize your image size, posts per page, image position, query (order by, select category, term, order, and get posts from. You can even customize the layout separately on other devices. This addon gives you full power to control your posts. Deen is super user-friendly and optimized for mobile first design. This addon includes all the features you need to make perfect and responsive posts for your website. This addon will be upgraded with more layouts and options to become the world's best post plugin. We will be adding more post layouts and options soon.

== Features ==

* Show the blog posts with 4 layout styles
* Customize layout styles options easily
* Display or hide post pagination 
* Post column can be controlled however you like
* Easily changeable image radius and space between image to content
* With our image CSS filter options, you can customize the image background color
* Show or hide Blog post titles, Excerpt, Comment, Button, Term
* Easily customizable Tabs color, Typography color and size, Term color and size
* Change pagination design, text, color and space as you like
* Add dynamically tags according to your posts
* Add dynamic categories based on posts
* Customize posts order easily


== Installation ==

Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You can type "Deen" on your element tabs within Elementor editor and all the avialable elements will appear.
4. Also you can see them under the category "Deen Posts" on your element/widget list.
